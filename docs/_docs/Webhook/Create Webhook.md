# `createWebhook`

```php
$client->webhook->createWebhook($parameters);
```

## Description

Create a new webhook.

## Parameters


Name | Type | Required | Default
--- | --- | --- | ---
name | string | false | *null*
avatar | string | false | *null*
channel.id | snowflake | true | *null*

## Response

Returns a webhook object on success.

Can Return:

* webhook
