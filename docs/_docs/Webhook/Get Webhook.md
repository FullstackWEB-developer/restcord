# `getWebhook`

```php
$client->webhook->getWebhook($parameters);
```

## Description



## Parameters


Name | Type | Required | Default
--- | --- | --- | ---
webhook.id | string | false | *null*

## Response

Returns the new webhook object for the given id.

Can Return:

* webhook
